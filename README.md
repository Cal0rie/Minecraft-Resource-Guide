# Minecraft Resource Guide

## 仓库介绍

一些自己收录的 Minecraft 资源。

## 仓库说明

### 模组篇

- 尽可能地收录较为优质且合我胃口的模组。
- 收录的模组我或多或少都有接触过或了解过。
- 不会收录主观上我不喜欢的模组。（如虚无世界和 ProjectE）

## ISSUE/PR

要是你和我一样闲着没事干，或者有出错的地方，可以新建 Issue，也可以直接发起 PR。
