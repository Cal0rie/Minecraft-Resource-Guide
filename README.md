# ModList

## 介绍

一些自己收录的 Minecraft 模组。

## 说明

- 尽可能地收录较为优质且合我胃口的模组。
- 收录的模组我或多或少都有接触过或了解过。
- 不会收录主观上我不喜欢的模组。（如虚无世界、神圣 RPG 和 ProjectE）
