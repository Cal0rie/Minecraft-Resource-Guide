## 收录说明

- 尽可能收录较为优质且合我胃口的模组
- 收录的模组我或多或少都有了解或接触过
- 分类方式参考 MCBBS / MCMOD 的模组分类
- 不太可能收录主观上我**非常不喜欢**的模组
